# 交易回測系統 - 使用說明

## 啟動方式

```bash
cd E:/ClaudeCode/Trade
pip install -r requirements.txt
python app.py
```

瀏覽器開啟 http://127.0.0.1:8050

---

## 系統架構

系統分為三個主要功能頁籤：

| 頁籤 | 功能 |
|------|------|
| 資料下載 | 下載個股/期貨歷史資料並匯出 CSV |
| 選股 | 透過基本面指標篩選標的 |
| 回測 | 設定技術分析策略並執行回測 |

---

## 一、資料下載

### 資料來源

| 來源 | 說明 | 標的範例 |
|------|------|---------|
| yfinance | 美股、期貨、加密貨幣、指數 | `AAPL`, `ES=F`, `BTC-USD`, `^GSPC` |
| TWSE 台灣證交所 | 台灣上市股票 | `2330` (台積電) |
| TPEx 櫃買中心 | 台灣上櫃股票 | `6547` |

### 操作步驟

1. **選擇資料來源** - 點選 yfinance / TWSE / TPEx
2. **輸入標的** - 從快速選擇清單中選擇，或手動輸入代碼
3. **設定日期範圍** - 選擇起訖日期
4. **選擇 K 線週期** (僅 yfinance)：1分/5分/15分/30分/1小時/日/週/月
5. **預覽資料** - 點擊「預覽資料」查看 K 線圖和資料表
6. **下載 CSV** - 點擊「下載 CSV」，檔案自動儲存至專案 `data/csv/` 資料夾，右上角顯示完成提示 (3 秒後自動消失)
7. **送入回測** - 點擊「送入回測 →」直接將已下載資料帶入回測頁

### 快速選擇清單

**期貨：**
- ES=F (S&P 500 E-mini)、NQ=F (Nasdaq 100)、YM=F (Dow Jones)
- GC=F (黃金)、CL=F (原油 WTI)、SI=F (白銀)
- 6E=F (歐元)、6J=F (日圓)、BTC-USD (比特幣)

**台股常用：**
- 2330 台積電、2317 鴻海、2454 聯發科、2882 國泰金 等

## 二、選股篩選

透過基本面財務指標篩選股票，找出符合條件的標的。

### 股票池來源

| 選項 | 說明 |
|------|------|
| 美股範例 (30檔) | 預設 S&P 500 中 30 檔常見股票 |
| 自訂清單 | 手動輸入股票代碼，以逗號分隔 |
| CSV 匯入 | 上傳包含基本面資料的 CSV 檔案 |

### 支援的基本面指標

| 指標 | 欄位名 | 說明 |
|------|--------|------|
| 本益比 | PE | 股價 / 每股盈餘 |
| 股價淨值比 | PB | 股價 / 每股淨值 |
| 殖利率 | DividendYield | 年度股利 / 股價 |
| ROE | ROE | 股東權益報酬率 |
| 營收成長率 | RevenueGrowth | 年營收成長率 |
| EPS | EPS | 每股盈餘 |

### 操作步驟

1. **選擇股票池** - 美股範例 / 自訂清單 / CSV 匯入
2. **設定篩選條件** - 最多可設定 3 個條件 (AND 邏輯)
   - 例：PE < 20、ROE > 15%、殖利率 > 3%
3. **點擊「篩選」** - 系統會從 yfinance 抓取基本面資料並套用條件
4. **查看結果** - 結果表格可排序，勾選感興趣的標的
5. **送入回測** - 勾選標的後點擊「送入回測 →」，自動跳轉至回測頁

### CSV 匯入格式

CSV 檔案需包含 `Symbol` 欄位，其餘欄位可為：
```
Symbol, Name, PE, PB, DividendYield, ROE, RevenueGrowth, EPS
```
也支援中文欄位名：`代碼, 名稱, 本益比, 股價淨值比, 殖利率, 股東權益報酬率, 營收成長率, 每股盈餘`

## 三、回測

### 資料來源設定

| 選項 | 說明 |
|------|------|
| 國際市場 (yfinance) | 輸入代碼即時下載，如 AAPL、ES=F |
| 已下載資料 | 從專案 `data/csv/` 資料夾讀取已下載的 CSV 檔案 |
| CSV 匯入 | 上傳自備的 OHLCV CSV 檔案 |

### 策略設定

#### 方向
- **做多 (Long)** - 買入後等待漲價賣出
- **做空 (Short)** - 賣出後等待跌價回補

#### 均線濾網 (進場前置條件)

在進場條件之前，可啟用均線濾網作為方向過濾器。濾網不通過時，即使進場條件全部滿足也不會進場。

| 濾網 | 說明 | 範例 |
|------|------|------|
| **均線位置濾網** | 收盤價在 MA/EMA 之上才能做多，之下才能做空 | 啟用 MA 20 → 價格在 20MA 上方才允許做多進場 |
| **均線方向濾網** | MA/EMA 方向持續上彎才能做多，下彎才能做空 | 啟用 MA 20 → 20MA 斜率為正才允許做多進場 |

- 兩個濾網可同時啟用 (AND 邏輯)
- 均線類型可選 MA (簡單移動平均) 或 EMA (指數移動平均)
- 週期可自由調整 (預設 20)
- 啟用後，濾網 MA 會顯示在 K 線圖上 (虛線)

#### 進場條件 (AND 邏輯 - 全部滿足才進場)

每條規則由 5 個部分組成：

| 欄位 | 說明 | 範例 |
|------|------|------|
| 指標 | 選擇技術指標 | RSI, MACD, MA, EMA, KD, Bollinger, ATR |
| 欄位 | 指標的輸出值 | RSI → `RSI`；MACD → `MACD_line` / `MACD_signal` / `MACD_hist`；KD → `K` / `D`；Bollinger → `BB_upper` / `BB_middle` / `BB_lower` |
| 參數 | 指標計算參數 (可留空用預設值) | `14` 表示 period=14；`fast=12` 指定特定參數 |
| 條件 | 比較運算子 | `>`, `<`, `>=`, `<=`, `crosses_above`, `crosses_below` |
| 值 | 門檻值或另一個欄位名 | 數字如 `30`，或欄位名如 `MACD_signal` |

點擊「+」按鈕可新增多條進場規則。

#### 出場條件 (OR 邏輯 - 任一滿足即出場)

設定方式與進場條件相同，但任何一條規則觸發就會執行出場。

### 技術指標參數預設值

| 指標 | 預設參數 | 輸出欄位 |
|------|---------|---------|
| MA (簡單移動平均) | period=20 | MA |
| EMA (指數移動平均) | period=20 | EMA |
| RSI (相對強弱指標) | period=14 | RSI |
| MACD (移動平均收斂擴散) | fast=12, slow=26, signal=9 | MACD_line, MACD_signal, MACD_hist |
| KD (隨機指標) | k_period=14, d_period=3 | K, D |
| Bollinger (布林通道) | period=20, std_dev=2.0 | BB_upper, BB_middle, BB_lower |
| ATR (平均真實範圍) | period=14 | ATR |
| Pivot (樞軸點) | pivot_type=1 | Pivot_P, Pivot_R1, Pivot_S1, Pivot_R2, Pivot_S2, Pivot_R3, Pivot_S3 |
| SRLevel (支撐壓力) | order=5, merge_pct=1 | SR_resistance, SR_support |

#### Pivot 樞軸點 — 參數說明

用前一根 K 棒的最高、最低、收盤計算當根 K 棒的支撐壓力水平。

| pivot_type | 類型 | 特點 |
|-----------|------|------|
| 1 | Standard (Floor) | 經典公式，P=(H+L+C)/3 |
| 2 | Fibonacci | 基於 Pivot 加上 0.382/0.618 倍的前一根振幅 |
| 3 | Camarilla | 基於收盤價加減振幅乘數，適合日內交易 |
| 4 | Woodie | 加重收盤價，P=(H+L+2C)/4 |

輸出 7 條水平線：P (樞軸)、R1/R2/R3 (壓力)、S1/S2/S3 (支撐)。

#### SRLevel 支撐壓力 — 參數說明

透過局部極值偵測找出歷史支撐壓力位，對每根 K 棒回傳距離最近的支撐和壓力。

| 參數 | 預設 | 說明 |
|------|------|------|
| order | 5 | 前後各幾根 K 棒判斷是否為局部極值 (越大越嚴格) |
| merge_pct | 1 | 合併容差 (%)，相距 1% 內的極值合併為一個水平 |

- **SR_resistance**: 當前收盤價上方最近的壓力位
- **SR_support**: 當前收盤價下方最近的支撐位
- 不偷看未來資料，僅使用已出現過的歷史極值

### 風控設定

| 參數 | 預設值 | 說明 |
|------|--------|------|
| 停損 % | 無 | 虧損達此比例自動平倉 |
| 停利 % | 無 | 獲利達此比例自動平倉 |
| 部位比例 % | 100 | 每次交易使用多少比例的資金 |
| 初始資金 | 1,000,000 | 回測起始金額 |
| 手續費 % | 0.1 | 每筆交易的手續費率 |
| 滑價 % | 0.05 | 模擬實際成交與理論價格的滑價 |

### 回測結果

回測完成後可查看三個子頁籤：

#### Chart (圖表)
- **K 線圖** - 顯示標的價格走勢
- **指標疊加** - MA、EMA、布林通道直接疊在 K 線上
- **進出場標記** - 綠色三角 (買入)、紅色三角 (放空)、橘色 X (平倉)
- **子圖** - RSI、MACD、KD、ATR 各自獨立顯示
- **成交量** - 底部顯示成交量柱狀圖
- **權益曲線** - 資金變化走勢 + 回撤百分比

#### Trades (交易明細)
- 每筆交易的進出場時間、價格、數量、損益、報酬率
- 獲利交易顯示綠色，虧損交易顯示紅色
- 可排序欄位

#### Metrics (績效指標)
| 指標 | 說明 |
|------|------|
| 總報酬率 | 整段回測期間的總報酬 |
| 年化報酬率 | 換算為年度報酬 |
| Sharpe Ratio | 風險調整後報酬 (越高越好) |
| 最大回撤 | 期間內最大虧損幅度 |
| 勝率 | 獲利交易占比 |
| 獲利因子 | 總獲利 / 總虧損 |
| 交易次數 | 總交易筆數 |
| 平均報酬 | 每筆交易的平均報酬率 |
| 做多/做空次數 | 各方向交易筆數 |

## 常見策略範例

### 1. RSI 超賣反彈 (做多)
- **進場**: RSI / RSI / 14 / < / 30
- **出場**: RSI / RSI / 14 / > / 70
- **停損**: 5%、**停利**: 10%

### 2. 均線黃金交叉 (做多)
- **進場**: EMA / EMA / 5 / crosses_above / EMA (需搭配 period=20 的 EMA)

### 3. MACD 線上穿信號線 (做多)
- **進場**: MACD / MACD_line / (預設) / crosses_above / MACD_signal
- **出場**: MACD / MACD_line / (預設) / crosses_below / MACD_signal

### 4. KD 超賣金叉 (做多)
- **進場條件 1**: KD / K / (預設) / crosses_above / D
- **進場條件 2**: KD / K / (預設) / < / 20
- **出場**: KD / K / (預設) / > / 80

### 5. 布林通道下軌反彈 (做多)
- **進場**: Bollinger / BB_lower / (預設) / > / (收盤價跌破下軌)
- **出場**: Bollinger / BB_upper / (預設) / < / (收盤價突破上軌)

### 6. RSI 超賣 + 均線濾網 (做多)
- **均線位置濾網**: 啟用，MA 60 (收盤價必須在 60MA 之上)
- **均線方向濾網**: 啟用，MA 20 (20MA 必須持續上彎)
- **進場**: RSI / RSI / 14 / < / 30
- **出場**: RSI / RSI / 14 / > / 70
- **說明**: 只在多頭趨勢中 (價格在長期均線之上 + 短期均線上彎) 做超賣反彈

### 7. MACD 金叉 + 均線方向濾網 (做多)
- **均線方向濾網**: 啟用，EMA 20 (EMA20 必須上彎)
- **進場**: MACD / MACD_line / (預設) / crosses_above / MACD_signal
- **出場**: MACD / MACD_line / (預設) / crosses_below / MACD_signal
- **說明**: 只在均線上彎的趨勢中做 MACD 金叉進場，避免在盤整或下跌趨勢中被騙

### 8. Pivot 樞軸點 S1 反彈 (做多)
- **進場**: Pivot / Pivot_S1 / (預設) / > / Close
- **出場**: Pivot / Pivot_R1 / (預設) / < / Close
- **說明**: 價格跌到 S1 支撐位附近時做多，漲到 R1 壓力位時出場

### 9. 支撐壓力突破 (做多)
- **進場**: SRLevel / SR_resistance / order=10 / crosses_above / SR_resistance
- **說明**: 收盤價突破局部極值壓力位時進場，order=10 可找出較大級別的壓力

## 完整操作流程範例

### 流程 A：選股 → 回測
1. 「選股」頁 → 選擇美股範例 → 設定 PE < 25, ROE > 10%
2. 點擊「篩選」→ 勾選 AAPL → 點擊「送入回測 →」
3. 「回測」頁 → 設定 RSI < 30 進場、RSI > 70 出場
4. 設定停損 5%、初始資金 1,000,000
5. 點擊「執行回測」→ 查看結果

### 流程 B：下載資料 → 回測
1. 「資料下載」頁 → 選擇 yfinance → 輸入 ES=F (S&P 500 期貨)
2. 設定日期 2024-01-01 ~ 2025-12-31 → 點擊「預覽資料」
3. 確認資料正確 → 點擊「下載 CSV」保存備份 (儲存至 `data/csv/`)
4. 點擊「送入回測 →」→ 自動跳轉回測頁
5. 資料來源自動切換為「已下載資料」，自動選取對應檔案 → 設定策略 → 執行回測

> **提示**：也可直接在回測頁選擇「已下載資料」，從下拉選單中選取 `data/csv/` 資料夾內任何已存在的 CSV 檔案

### 流程 C：台股回測
1. 「資料下載」頁 → 選擇 TWSE → 輸入 2330 (台積電)
2. 設定日期範圍 → 點擊「預覽資料」→「送入回測 →」
3. 「回測」頁 → 設定策略 → 執行回測

## 注意事項

1. **yfinance 資料限制**：分鐘級資料最多回溯 60 天，日線無限制
2. **台股資料速率**：TWSE/TPEx 下載會自動限速 (每月 0.5 秒)，長期間資料需等待
3. **crosses_above / crosses_below**：比較前後兩根 K 棒的值，判斷是否發生交叉
4. **回測執行**：逐 K 棒模擬，以下一根 K 棒的開盤價成交，已計入手續費和滑價
5. **同時只持有一個部位**：不支援加碼或同時持有多個方向的部位
